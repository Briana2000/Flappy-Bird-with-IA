# Flappy-Bird-with-IA
En este proyecto, abordaremos el desafío de entrenar un agente inteligente para jugar al juego de Flappy Bird
utilizando técnicas de aprendizaje por refuerzo (reinforcement learning) y redes neuronales convolucionales.
Utilizaremos el entorno de Flappy Bird proporcionado por https://github.com/robertoschiavone/flappybird-env.
